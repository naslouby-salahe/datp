## Goal

Freeze all result artifacts and verify the full lineage chain. Produce the frozen result package that gates manuscript writing.

## Why This Exists

Manuscript editing starts only after result freeze. This ensures no unsupported claim enters the paper and all results have valid provenance.

## Scope

- Verify all active experiments completed or explicitly suppressed
- Verify all reused cells are `VERIFIED_REUSE_SAFE`
- Verify all metrics reproducible
- Same-seed determinism verification
- Resolved configs exist for every run
- Lineage exists for every reported result
- Generate all tables/figures from canonical reporting pipeline
- Verify figure sidecars exist
- Document null, mixed, failed, and skipped experiments
- Anti-HARKing attestation: compare planned claims against locked wording

## Out of Scope

- Running new experiments (freeze means freeze)
- Manuscript writing (that is T27)
- Reopening results after freeze without explicit process
