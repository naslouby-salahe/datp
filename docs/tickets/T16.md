## Goal

Overlay benign/attack CDFs with B1/B2/B4 threshold lines per client. Produce CDF figure and failure-mode table explaining worst-client behavior.

## Why This Exists

Provides visual mechanistic explanation of how thresholds interact with per-client score distributions. Essential for explaining why some clients benefit more from personalization.

- Overlay B1, B2, B4 threshold values as vertical lines
- Identify failure modes: clients where B1 causes high FPR or low TPR
- Produce CDF overlay figure (per device or summary)
- Produce failure-mode table
