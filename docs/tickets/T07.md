## Goal

Implement the τ-shrink interpolation: `τ_k(λ)=λτ_k+(1−λ)τ_global` for `λ ∈ {0, 0.25, 0.5, 0.75, 1}`. Produce the full λ curve. Verify endpoint equivalence: λ=0 → B1, λ=1 → B2.

## Why This Exists

τ-shrink explores whether intermediate interpolation between global and per-client thresholds can reduce disparity without harming low-end clients. The full λ curve is the result — no single λ is selected post-hoc as "optimal."

## Scope

- Compute τ_global (B1 threshold) and τ_k (B2 per-client thresholds)
- Interpolate at each λ value
- Evaluate metrics at each λ
- Produce λ curve with CV(FPR), Macro-F1, P10 Macro-F1
- Verify λ=0 reproduces B1, λ=1 reproduces B2

## Out of Scope

- Selecting an "optimal" λ
- Tuning λ after results are seen
- Retraining
