## Goal

Extract verified MAC/group/client metadata from raw CICIoT2023 CSV files, define B-b clients, and generate train/cal/test splits if metadata supports it.

## Why This Exists

Phase 0 GA-07 audit found: processed files contain only 39 feature columns, no MAC/device/group fields. Raw CSV files are needed to determine if B-b is feasible. Currently `B_B_BLOCKED_UNVERIFIED`.

## Scope

- Obtain raw CICIoT2023 CSV files
- Inspect for MAC, source/destination device, group, capture-source columns
- Determine B-b feasibility outcome
- If feasible: define clients, validate eligibility (≥8 clients, n_cal≥100 each)
- Generate B-b processed splits (Parquet)

- Modifying existing Regime B-a processing
