## Goal

Evaluate B1/B2/B4 thresholds and metrics at `q ∈ {0.90, 0.95, 0.975, 0.99}` using verified stored scores. Produce a per-q table and heatmap. Confirm q=0.95 reproduces reference values.

## Why This Exists

The journal must demonstrate that the B1→B2 CV(FPR) effect is not an artifact of the single q=0.95 operating point. This is a reviewer-defense analysis.

## Scope

- Iterate over q grid: `{0.90, 0.95, 0.975, 0.99}`
- For each q, recompute B1/B2/B4 thresholds and metrics on verified Regime A scores
- Conditional: apply to Regime C and Regime D if verified scores exist
- Produce q-sensitivity table (CSV/JSON)
- Produce heatmap figure

## Out of Scope

- Retraining at different q values
- Adding new q values post-result inspection
- Regime D/B-b (unless their scores exist and are verified)
