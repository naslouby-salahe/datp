## Goal

Apply the claim-survival matrix to frozen results, select wording per outcome tables, and update the manuscript in the prescribed order: Results → Methods → Discussion → Limitations → Related Work → Abstract → Conclusion → Supplement.

## Why This Exists

Manuscript claims must follow frozen numbers. The claim-survival matrix in POST_EXPERIMENT_PLAN §2 maps every possible outcome to specific required wording. This ensures no overclaiming.

## Scope

- Apply claim-survival matrix to all frozen result areas
- Select locked wording for each outcome
- Update manuscript sections in order
- Include conference-extension disclosure
- Include B-LaridiFaithful locked disclosure sentence
- Apply page-budget cuts if needed (per POST_EXPERIMENT_PLAN §5.3)
- Main paper vs supplement placement per §5.1/§5.2 rules

## Out of Scope

- Running new experiments
- Changing frozen numbers
- Adding claims not supported by results
- Tuning parameters after results are seen
